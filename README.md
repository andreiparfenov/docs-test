# TypeScript for Beginners

A short, 5-lesson course that takes you from "what is TypeScript?" to writing typed classes and interfaces. Each lesson is a standalone Markdown file with explanations, code examples, and a short exercise.

## Lessons

1. [Getting Started with TypeScript](lessons/01-getting-started.md) — what TypeScript is, installing it, and compiling your first file.
2. [Basic Types](lessons/02-basic-types.md) — `string`, `number`, `boolean`, arrays, tuples, `any`, and type inference.
3. [Functions and Interfaces](lessons/03-functions-and-interfaces.md) — typing function parameters/return values and describing object shapes with `interface`.
4. [Classes](lessons/04-classes.md) — classes, access modifiers, and inheritance in TypeScript.
5. [Generics](lessons/05-generics.md) — writing reusable, type-safe functions and structures with generics.

## Prerequisites

Basic familiarity with JavaScript (variables, functions, objects, arrays) is assumed. No prior TypeScript experience is required.

## How to use this course

Each lesson includes runnable code snippets. To try them yourself:

```bash
npm install -g typescript
tsc yourfile.ts
node yourfile.js
```

Work through the lessons in order — each one builds on the previous.
